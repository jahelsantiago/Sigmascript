# Sigmascript
Sigmascript is a programming language 
It is a special purpose programming language for the two-way classification problem using the concepts of runs. The language was built under the imperative paradigm (which includes sequence, selection and iteration structures) and the possibility of defining and managing variables, arrays and arrays.

It was build using [PLY (Python lex yacc)](https://www.dabeaz.com/ply/ply.html#ply_nn4)  

## Modification Requirments
1. Clone this repository 
2. Install numpy
3. Open the calc.py file and run it
